# Desafio Tree Tecnologia
Um projeto ASP.NET Core (v3.1) Web API que consome a SWAPI.dev.  Esse projeto implementa o Swagger.

# Visual Studio
Abra o arquivo da solução <code>DesafioTree.sln</code> e dê um build/run.

# Swagger
Para explorar e testar a API simplismente compile o projeto e use o Swagger UI.

Endpoints disponíveis:
- GET `/api/people` - Retorna todas as pessoas listadas na SWAPI.dev 
- GET `/api/planets` - Retorna todos os planetas listados na SWAPI.dev 
- GET `/api/starships` - Retorna todas as espaço-naves listadas na SWAPI.dev 
- GET `/api/travels/number-stops{distance}` - Retorna em uma lista quantas paradas cada espaço deverá fazer para uma determinada distância
- GET `/api/people/best-starship` - Retorna qual a melhor espaço-nave para um determinado número de pessoas e uma distância.
